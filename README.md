# Siddhesh Nimodiya — Developer Portfolio

A modern, animated, and fully responsive portfolio built with **React**, **TailwindCSS**, **GSAP**, and **React-Three-Fiber**.
This website showcases my skills, journey, projects, and interactive 3D visuals.

---

## Features

* Fast and modern UI built with Vite + React
* Smooth animations using GSAP + ScrollTrigger
* 3D interactive skill icons built with React-Three-Fiber
* Fully responsive across all screen sizes
* Clean experience timeline and skill sections
* Reusable components and organized folder structure

---

## Tech Stack

Frontend: React, JavaScript (ES6+), Vite
Styling: TailwindCSS
Animations: GSAP, ScrollTrigger
3D Models: React-Three-Fiber, Drei, Three.js
Assets: Custom GLB models and optimized images
